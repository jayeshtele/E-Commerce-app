# NovaCart E-Commerce App

NovaCart is a frontend-only e-commerce web app built with React, Vite, React
Router, Redux Toolkit, Tailwind CSS, and ESLint.

## Features

- Live product catalog from the DummyJSON products API
- Amazon-style multi-page shopping flow
- Home, catalog, category, deals, product detail, cart, checkout, and success pages
- Product thumbnails, image galleries, ratings, stock labels, discounts, shipping,
  warranty, returns, dimensions, tags, and reviews
- Search, category filtering, price filtering, stock filtering, and sorting
- Redux-powered cart with persisted local storage
- Checkout confirmation with estimated shipment dates

## Run Locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Quality Checks

```bash
npm run lint
npm run build
```
